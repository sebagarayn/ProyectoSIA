NOTAS DEL PROYECTO - SISTEMA VETERINARIA
========================================

ESTADO ACTUAL:
- Sistema base implementado y funcional.
- Estructura de clases definidas, cada una con atributos privados (Veterinaraia,
Cliente, Mascota, Servicio.)
- Funcionalidades principales como agregear (Cliente, Mascota, Servicio) y
mostrar.
- Contiene una anidación con la siguiente jerarquia: 
Veterinarai->Cliente->Mascota->Servicio.
- Se incluye un menu para realizar las funciones integradas.
- Se utiliza sobrecarga de metodos.
- Incluye datos iniciales dentro del codigo.
- Incluye un HashMap mediante el uso del rut de cada cliente.
